  <action>
Refactor install.js line 286. Replace the bare `main();` call with:

```javascript
if (require.main === module) {
  main();
}

module.exports = {
  main,
  detectRuntime,
  validateInstall,
  copyDirSync,
  dirExists,
  fileExists,
  printResults,
  DIRS_TO_COPY,
  FILES_TO_COPY,
};
```

Per D-01: The `require.main === module` guard ensures that when another file does `require('./install.js')`, the `main()` function does NOT execute. When Node.js runs the file directly (via `node install.js` or `npx taketomarket`), `require.main === module` is true and `main()` runs as before.

Per D-02: Export these specific functions: detectRuntime, validateInstall, copyDirSync, dirExists, fileExists, printResults. Also export main (for completeness, per RESEARCH.md recommendation) and the constants DIRS_TO_COPY, FILES_TO_COPY.

Per D-03: Do NOT modify any function bodies. Do NOT remove process.exit() calls from main(). The only change is replacing line 286 `main();` with the guard block + module.exports block. This preserves backwards compatibility.

Do NOT add any new imports or modify any existing function definitions. The entire change is at the bottom of the file only.
  </action>
